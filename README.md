# HackTheBox Writeups

My cybersecurity learning journey through Hack The Box.

This repository contains my personal notes, methodology, enumeration process, commands, and lessons learned while solving Hack The Box machines.

The goal is not only to capture flags, but to understand how systems work, identify attack surfaces, and develop a structured penetration testing methodology.

---

## Learning Objectives

Through these machines, I aim to strengthen my understanding of:

- Network Enumeration
- Service Discovery
- Linux Fundamentals
- Windows Fundamentals
- Web Security
- Database Security
- Authentication & Access Control
- Active Directory
- Privilege Escalation
- Defensive Security Concepts
